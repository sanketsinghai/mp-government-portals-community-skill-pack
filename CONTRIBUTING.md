# Contributing

## What to contribute

Contributions are welcome for:
- Broken or outdated official URLs
- Missing MP government portals
- Better categorization of existing portals
- Documentation improvements
- Additional example prompts or setup guidance

## Contribution rules

- Only add official government portals or clearly official government-operated service portals.
- Prefer `gov.in`, `nic.in`, `mp.gov.in`, or other official state-controlled domains.
- Do not add commercial, unofficial, or speculative links.
- Keep descriptions short, factual, and neutral.
- Do not guess URLs. If a portal cannot be verified, leave it out.

## Before opening a pull request

1. Confirm the link is official and currently reachable.
2. Place the entry in the correct reference file.
3. Keep the existing formatting style of the table or section.
4. If a portal moved, replace the old URL and mention the change in the pull request.
5. If a new category is needed, explain why in the pull request.

## Pull request checklist

- State what changed
- Explain how the URL was verified
- Mention whether an old URL was removed or redirected
- Note any district, department, or service impact

## Issues

Open an issue if:
- A portal is broken
- A portal has changed domain
- A district or department is missing
- Installation or usage instructions are unclear

## Maintenance expectation

This repository is curated manually. Fast fixes are possible, but contributors should assume human review before merge.
